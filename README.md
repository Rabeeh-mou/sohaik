# Sohaik T-Shirts E-Commerce Website

## Project Description
Sohaik T-Shirts is a modern e-commerce website developed using React. The website allows users to browse and shop for different t-shirts through a clean and responsive interface.

Users can:
- View products
- Add items to the shopping cart
- Manage a wishlist
- Browse products easily on desktop and mobile devices

The project demonstrates modern frontend web development concepts including reusable React components, routing, responsive design, and state management.

---

# Features
- Home Page
- Product Listing Page
- Shopping Cart
- Wishlist
- Responsive Design
- Modern User Interface
